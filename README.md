##Job Matching & Filtering System

A web-based application that helps users find and filter job listings based on their preferences such as job position, 
location, work arrangement, job type, and salary range. This project was developed as part of a school requirement using HTML and CSS.

##Live Demo Live Website: https://astounding-tartufo-bd8b9a.netlify.app/

##Features

1. Job filtering system (Job position, location, job type, salary range, and work arrangement)
2. User registration and login system
3. Job listings display
4. Applications (Users can view all their applications)
5. Schedules (Users can view all their interview schedules)
6. Help/Support
7. Profile
